# Tribia Futbolera

**Demuestra tu conocimiento del fútbol. Solo humanos únicos.**

Tribia Futbolera es una Mini App de World App diseñada exclusivamente para humanos verificados que quieren poner a prueba su capacidad de análisis futbolístico y conocimiento del deporte.

## 🎯 ¿Qué es Tribia Futbolera?

Un juego de habilidad y predicción donde analizas estadísticas, formas de equipos, historial y tendencias para predecir los resultados del Mundial 2026. No es suerte, es conocimiento.

### Solo para Humanos Únicos
- Verificación World ID requerida
- Un humano = Una cuenta
- Competencia justa basada en habilidad

## 🧠 Psicología del Juego

Tribia Futbolera está diseñada para:
- **Análisis estratégico**: Estudia estadísticas, rendimiento de equipos, lesiones, clima
- **Toma de decisiones informadas**: Usa tu conocimiento del fútbol
- **Pensamiento anticipado**: Las predicciones tempranas tienen mayores recompensas
- **Gestión de riesgo**: Decide cuándo y qué predecir

No es azar. Es tu capacidad de leer el juego.

## 💎 Sistema de Recompensas WGoal

### Recompensas Diarias
- **1 WGoal** por participación diaria activa
- Mantén tu racha para bonificaciones

### Bonos por Predicciones Anticipadas
Mientras más temprano predices correctamente, mayor la recompensa:
- Predicción pre-torneo: **Bono x5**
- Predicción fase de grupos: **Bono x3**
- Predicción octavos: **Bono x2**

### Premios Mayores
- **Ganador Copa del Mundo**: 1,000,000 WGoal
- **Top 3 Final**: 500,000 WGoal
- **Semifinalistas correctos**: 100,000 WGoal
- **Fase eliminatoria perfecta**: 50,000 WGoal

## 🎮 Cómo Funciona

1. **Verifica tu humanidad** con World ID
2. **Analiza** estadísticas y datos de equipos
3. **Predice** resultados basándote en tu conocimiento
4. **Gana WGoal** por predicciones correctas
5. **Compite** en el ranking global de analistas

## 🛡️ Cumplimiento y Políticas

- ✅ **No es apuesta**: Es un juego de habilidad y conocimiento
- ✅ **Verificación humana**: World ID garantiza un humano por cuenta
- ✅ **Recompensas por mérito**: Basadas en análisis y conocimiento
- ✅ **Transparente**: Todas las predicciones son registradas on-chain
- ✅ **Educativo**: Fomenta el análisis estadístico y pensamiento crítico

## 🚀 Tecnología

- **World ID**: Verificación de humanidad única
- **Minikit**: Integración nativa con World App
- **Next.js 15**: Framework moderno y rápido
- **Blockchain**: Registro transparente de predicciones

## 📱 Instalación para Desarrollo

```bash
# Instalar dependencias
npm install

# Configurar variables de entorno
cp .env.example .env.local

# Ejecutar en desarrollo
npm run dev

# Exponer con ngrok para testing
ngrok http 3000
```

## 🔧 Configuración

1. Actualiza `.env.local` con tus credenciales de World App
2. Genera `AUTH_SECRET` con `npx auth secret`
3. Configura tu app en [developer.worldcoin.org](https://developer.worldcoin.org)
4. Agrega tu dominio a `allowedDevOrigins` en `next.config.ts`
