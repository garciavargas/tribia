# Tribia - Worldcoin Mini App

Mini aplicación para el ecosistema de Worldcoin.

## 📁 Estructura del Proyecto

```
tribia/
├── app/                    # Aplicación Next.js
│   ├── layout.tsx         # Layout principal
│   ├── page.tsx           # Página principal
│   └── globals.css        # Estilos globales
├── public/                # Recursos estáticos
│   ├── pagina1.png
│   ├── pagina2.png
│   └── paginatoken.png
└── manual/                # 📚 DOCUMENTACIÓN COMPLETA DE WORLDCOIN
    ├── README.md                    # Índice de la documentación
    ├── PROTOCOLO_WORLDCOIN.md       # Protocolo completo
    ├── COMANDOS_DETALLADOS.md       # Guía de comandos
    ├── GUIA_INICIO_RAPIDO.md        # Tutorial paso a paso
    └── MEJORES_PRACTICAS.md         # Mejores prácticas
```

## 🚀 Inicio Rápido

### Instalación

```bash
pnpm install
```

### Desarrollo

```bash
pnpm dev
```

### Build

```bash
pnpm build
```

## 📖 Próximos Pasos

1. Lee la [Guía de Inicio Rápido](./manual/GUIA_INICIO_RAPIDO.md)
2. Instala MiniKit siguiendo el [Protocolo](./manual/PROTOCOLO_WORLDCOIN.md)
3. Implementa funcionalidades usando [Comandos Detallados](./manual/COMANDOS_DETALLADOS.md)
